<h1>SONNIM</h1>

<h3>1) 개요</h3>
최근 여행의 인기 숙박 상품으로 떠오르는 게스트 하우스 전용 예약 플랫폼
</br>
</br>

<h3>2) 사용 기술 및 환경</h3>

- JAVA17
- SPRING BOOT 3.2.5
- SPRING DATA JPA
- AWS RDS (Maria DB)
- AWS EC2
- AWS ECR
- REDIS
- DOCKER
- GITHUB ACTIONS
